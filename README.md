
## Project Overview
Market Mate is a simple web application created to help mama mboga(small-scale vegetable and grocery sellers in Kenya that run roadside stalls or kiosks)manage their businesses with competentness by enabling them to track and stock, calculate their daily sales correctly with so much ease.It mainly assists them in reducing daily loses and improve profit tracking.

## Features
1. Add new products using a form
2. Form validation with error messages
3. Store data using local storage
4. Display saved products automatically
5. Responsive layout(mobile,tablet and desktop)
6. Multiple linked pages
7. Easy navigation
## Setup|Installation Requirements
1. Download this repository from GitHub.
2. Open the project folder using VS code.
3. Ensure the following exist: index.html,sales.html,about.html,add stock.html,style.css,mama mboga.js
## Technologies used
1. HTML for structring the web pages and semantics
2. CSS for styling, layout and responsive designs
3. JS for event handling,form validation and local storage
4. Git and GitHub for version control and local storage
## How to run the Project
1. Clone the repository:git clone https://github.com/mercymnn6-arch/Final-Project.git
2. Open the project folder Visual Studio Code.
3. Make sure you have a live Server extension installed in VS Code.
4. Right-click on the html files and select open with live server.
5. Use the navigation menu to move between pages.
